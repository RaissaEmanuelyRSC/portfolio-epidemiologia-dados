## Ferramentas
- R
- Python
- Power BI
- SQL
- QGIS

## Projetos

### 1. Análise Epidemiológica de Arboviroses
- Descrição: análise de dados de dengue, Zika e chikungunya
- Ferramentas: R, Power BI
- Atividades:

  - análise temporal e espacial
  - construção de indicadores
- Resultado:
  - identificação de padrões epidemiológicos
